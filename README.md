# darts-clone.rb

darts-clone.rb is Ruby bindings for the Darts-clone.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'darts-clone-rb'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install darts-clone-rb

## Usage

```ruby
require 'darts'

```
