# Darts-clone.rb

Darts-clone.rb is a Ruby binding for the [Darts-clone](https://github.com/s-yata/darts-clone).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'darts-clone-rb'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install darts-clone-rb

## Usage

```ruby
require 'darts'

```
